# ⚡ Pokémon UI Card - CSS Positioning Task

A pixel-perfect, responsive web UI clone of a modern Pokémon card design. This project was built to deeply understand and implement advanced CSS layout techniques. 

### 🚀 Live Demo
[Click here to view the live project](YAHAN_APNA_LIVE_LINK_DAAL_DENA)

### 🛠️ Tech Stack
* **HTML5:** Semantic structure and layout.
* **Vanilla CSS3:** Custom styling, Flexbox, and Media Queries.
* **Remix Icons:** For scalable UI icons.
* **Google Fonts:** Utilizing 'Poppins' for a clean, modern typography.

### 🧠 Key Learnings & Concepts Applied
* **Advanced Positioning:** Mastered the interplay between `position: relative` and `position: absolute` to break elements out of the normal document flow (e.g., the overlapping Pikachu hero image).
* **Flexbox Layouts:** Used for aligning the main container and structural child elements perfectly in the center of the viewport.
* **Responsive Web Design:** Implemented multiple CSS `@media` queries to ensure the UI gracefully adapts across Desktop, Tablet, and Mobile screens without breaking the layout.
* **Z-Index Management:** Controlled the stacking order of overlapping elements (text, background logo, and hero image) for a cohesive 3D effect.
* **CSS Filters:** Applied `mix-blend-mode` and image `saturate` filters to manipulate image assets directly in the browser.
